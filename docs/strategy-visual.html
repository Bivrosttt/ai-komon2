01 / PURPOSE

このサイトは何のため？

従業員10〜50名の中小企業経営者に、「AI人材を雇う前にAI顧問を選ぶ」理由を伝え、無料相談まで連れていく場所です。

→
業種向けページ「うち向けはここ」
→
デモを触る3本まで
→
事例を読む似た会社の変化
→
無料相談30分

06 / COMPARE

Before / After 一覧

項目 今（Before） これから（After）
